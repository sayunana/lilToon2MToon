# lilToon2MToon
lilToonをMToonに変換するエディターです。

# 使い方
メニューバーから `sayunana` -> `lilToon2MToon`でエディターを開きます。\
MToonに変換したMaterialを保存するフォルダーを選択します。\
アバターオブジェクトにアバターをセットします。\
`lilToonをMToonに変換する`を押すと変換されます。


# 注意
マテリアルの差し替えをします。\
バックアップを取って実行することをオススメします。
